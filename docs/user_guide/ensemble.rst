Ensemble generation
===================

``cht_cyclones`` implements a probabilistic ensemble approach based on
DeMaria et al. (2009), in which a set of perturbed tracks and wind fields is
generated by drawing random along-track, cross-track, and intensity errors
from auto-regressive processes calibrated on historical NHC forecast errors.

Overview
--------

A :class:`~cht_cyclones.ensemble.TropicalCycloneEnsemble` wraps a single
deterministic :class:`~cht_cyclones.tropical_cyclone_refactored.TropicalCyclone`
(the "best track", member 0) and generates *N* additional members, each with a
perturbed track geometry and intensity.  Optionally it also computes a scaled
spiderweb wind field for every member.

Creating an ensemble
---------------------

.. code-block:: python

   from datetime import datetime
   from cht_cyclones import TropicalCyclone
   from cht_cyclones.ensemble import TropicalCycloneEnsemble

   # Load a best-track
   tc = TropicalCyclone(track_file="idai_best_track.cyc")

   # Build the ensemble
   ens = TropicalCycloneEnsemble(
       tropical_cyclone=tc,
       name="idai_ensemble",
       number_of_realizations=50,
       compute_wind_fields=True,
       dt=3,                          # resample track to 3-h intervals
       tstart=datetime(2019, 3, 9),   # start of forecast period
       duration=120,                  # hours
       track_path="output/tracks",
       spw_path="output/spiderwebs",
       format="ascii",                # "ascii" or "netcdf"
   )

   # Generate tracks and (optionally) wind fields
   ens.generate()

After calling :meth:`~cht_cyclones.ensemble.TropicalCycloneEnsemble.generate`
the ``ens.members`` list contains ``number_of_realizations + 1`` entries
(member 0 = best track).

Error statistics
----------------

The default error statistics are based on NHC verification data for 2018–2021.
They can be overridden at construction time:

.. list-table::
   :header-rows: 1
   :widths: 30 15 55

   * - Parameter
     - Default
     - Description
   * - ``mean_abs_cte24``
     - 19.04 nm
     - Mean absolute 24-h cross-track error
   * - ``sc_cte``
     - 1.3253
     - Auto-regression coefficient for cross-track error
   * - ``mean_abs_ate24``
     - 26.22 nm
     - Mean absolute 24-h along-track error
   * - ``sc_ate``
     - 1.3432
     - Auto-regression coefficient for along-track error
   * - ``mean_abs_ve24``
     - 6.99 knots
     - Mean absolute 24-h intensity error
   * - ``sc_ve``
     - 1.0000
     - Auto-regression coefficient for intensity error
   * - ``bias_ve``
     - 0.0
     - Intensity bias per forecast hour (knots/h)

.. code-block:: python

   nm_to_m = 1852.0

   ens = TropicalCycloneEnsemble(
       tropical_cyclone=tc,
       number_of_realizations=100,
       mean_abs_cte24=22.0,          # nm
       sc_cte=1.40,
       mean_abs_ate24=30.0,          # nm
       mean_abs_ve24=8.0,            # knots
       bias_ve=0.05,
   )
   ens.generate()

Track resampling
----------------

Before perturbation the best track is resampled to equidistant time steps of
``dt`` hours using cubic-spline interpolation.  This ensures that the
auto-regressive error processes have a consistent temporal correlation length.
Shorter ``dt`` values produce smoother but more tightly clustered ensemble
members; larger values allow more spread.

Wind-field scaling
------------------

When ``compute_wind_fields=True`` each member's spiderweb is computed by
scaling the best-track wind field rather than running the full Holland (2010)
computation from scratch.  The scale factor is derived from the ratio of the
perturbed vmax to the best-track vmax at each time step, which is
computationally efficient for large ensembles.

Member files
------------

Track and spiderweb files for each member are written to ``track_path`` and
``spw_path`` respectively, named with the ensemble base name and a zero-padded
member index:

.. code-block:: text

   output/tracks/
   ├── idai_ensemble_00.cyc   # best track (member 0)
   ├── idai_ensemble_01.cyc
   ├── ...
   └── idai_ensemble_50.cyc

   output/spiderwebs/
   ├── idai_ensemble_00.spw
   ├── ...
